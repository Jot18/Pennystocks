How it works

A GitHub Actions workflow runs hourly Monday-Friday. It scans ~60 liquid penny/low-priced tickers via Yahoo Finance, fetches pre-market data, scrapes news from Yahoo Finance and Finviz, classifies headlines for catalyst keywords, and commits everything back to the repo as JSON. This page reads those JSON files directly — no backend server needed.

Scoring (heuristic, NOT a prediction)

  • Rel volume > 2× — +25
  • Gap > 5% — +15
  • RSI < 30 (oversold) — +15
  • RSI > 70 (overbought) — +10
  • Uptrend (price > MA20, MA5 > MA20) — +10
  • ATR > 8% (volatile) — +10

News classification

Headlines are tagged as bullish (FDA approval, contract win, beats estimates, buyback…), bearish (delisting, dilution, SEC investigation, missed estimates…), mixed, or neutral. This is keyword matching — it's a starting point, not analysis.

Honest disclaimers

  • No algorithm reliably predicts tomorrow's price. If one existed, it wouldn't be free on GitHub.
  • Penny stocks are favorite targets of pump-and-dump schemes. High score ≠ legitimate move.
  • Most retail day traders lose money over time. Studies put profitability under 20% over multi-year horizons.
  • This is educational software. Not financial advice.
